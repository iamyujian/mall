# mall

项目是使用 vue 开发的移动端单页面电商应用，使用webpack进行模块化处理。实现可以浏览商品、添加购物车等交互功能，并对项目进行了一些简单的优化和适配，算是对 vue 学习过程中的一个小的实践，感谢 coderwhy 大佬提供的项目接口。
